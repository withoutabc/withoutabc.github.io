# 岛屿数量

给你一个由 1 （陆地）和 0 （水）组成的的二维网格，请你计算网格中岛屿的数量。

关键步骤已经注释出：

  1. 按顺序遍历二维数组，每次遍历时通过 dfs 函数搜索其上下左右位置
  2. 防止出边界的情况，提前计算好长、宽，及时 return
  3. 对访问过的位置进行标记，防止重复访问导致无法退出递归
  4. 搜索中遇到非 1 处即停止，这样保证一次搜出了一整块陆地，且不会被重复计数
