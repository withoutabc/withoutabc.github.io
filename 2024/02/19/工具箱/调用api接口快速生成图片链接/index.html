# 上传图片

这里我们想写一个程序快速给图片批量上传图床。

  • upload_file 函数：传入要上传的图片文件夹的路径
  • Authorization 字段：填入登录获得的 token

免费版有限制：每分钟 20 次，每小时 100 次。

建议一次放十几张图片，比较合适。
